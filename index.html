📘 Lunes 15 de Junio - ¿Que es Excel y para que se usa?

Introduccion a Microsoft Excel y sus aplicaciones practicas
Objetivo del tema: Comprender que es Microsoft Excel, conocer sus principales usos y aplicaciones en el ambito academico, personal y profesional.

📖 Definicion de Microsoft Excel

Microsoft Excel es un programa de hojas de calculo desarrollado por Microsoft que forma parte de la suite Microsoft Office. Permite organizar, analizar y manipular datos en formato de tablas compuestas por filas y columnas.

Excel es una de las herramientas informaticas mas utilizadas en el mundo para el manejo de datos numericos y financieros. Su versatilidad lo convierte en una aplicacion indispensable en oficinas, escuelas, empresas y hogares.

Caracteristicas principales de Excel:

  • Hojas de calculo: Organizacion de datos en celdas, filas y columnas
  • Formulas y funciones: Calculos automaticos y complejos
  • Graficos: Representacion visual de datos
  • Tablas dinamicas: Resumen y analisis de grandes cantidades de datos
  • Macros: Automatizacion de tareas repetitivas
  • Formato condicional: Resaltado visual de datos segun criterios
  • Validacion de datos: Control de la informacion que se ingresa

¿Para que se usa Excel?

Excel tiene multiples aplicaciones en diferentes areas:

  1. Area academica:
    • Elaboracion de cuadros de notas y registros de calificaciones
    • Creacion de horarios y cronogramas de estudio
    • Organizacion de datos para proyectos de investigacion
    • Realizacion de calculos matematicos y estadisticos
    • Elaboracion de presupuestos para eventos escolares
  2. Area personal:
    • Control de gastos e ingresos personales
    • Elaboracion de presupuestos familiares
    • Planificacion de ahorros e inversiones
    • Organizacion de listas de compras y tareas
    • Seguimiento de habitos y metas personales
  3. Area profesional y empresarial:
    • Contabilidad y finanzas (estados financieros, balances)
    • Gestion de inventarios y stock de productos
    • Control de nomina y recursos humanos
    • Analisis de ventas y proyecciones comerciales
    • Elaboracion de facturas y cotizaciones
    • Gestion de proyectos y cronogramas
    • Creacion de bases de datos simples

Ventajas de usar Excel

  • Facilidad de uso: Interfaz intuitiva y amigable
  • Versatilidad: Se adapta a cualquier tipo de calculo o analisis
  • Precisión: Elimina errores de calculo manual
  • Rapidez: Realiza operaciones complejas en segundos
  • Organizacion: Permite ordenar y filtrar grandes cantidades de datos
  • Visualizacion: Crea graficos y tablas para entender mejor la informacion
  • Compatibilidad: Funciona en Windows, Mac, dispositivos moviles y web

📗 Martes 16 de Junio - Interfaz del Programa: Cinta de Opciones, Pestañas y Area de Trabajo

Conociendo los elementos visuales y herramientas de Excel
Objetivo del tema: Identificar y comprender cada elemento de la interfaz de Excel: la cinta de opciones, las pestañas, grupos, comandos y el area de trabajo.

📖 Elementos de la interfaz de Excel

La interfaz de Excel es el conjunto de elementos visuales con los que el usuario interactua para realizar sus tareas. Conocer cada parte de la interfaz es fundamental para trabajar de manera eficiente.

Partes principales de la interfaz:

  • Barra de titulo: Muestra el nombre del archivo y los botones de control (minimizar, maximizar, cerrar)
  • Cinta de opciones (Ribbon): Barra principal con todas las herramientas organizadas en pestañas
  • Pestañas: Categorias principales: Inicio, Insertar, Diseno de pagina, Formulas, Datos, Revision, Vista, Programador
  • Grupos: Conjuntos de comandos relacionados dentro de cada pestaña
  • Comandos: Botones y herramientas especificas para realizar acciones
  • Area de trabajo: La cuadricula donde se ingresan los datos (celdas, filas, columnas)
  • Barra de formulas: Muestra el contenido de la celda seleccionada y permite editar
  • Barra de estado: Informacion en la parte inferior (suma, promedio, zoom, etc.)
  • Pestañas de hojas: Permiten navegar entre diferentes hojas del libro

📓 COPIAR EN CUADERNO - Tema 2

La Cinta de Opciones (Ribbon)

La cinta de opciones es la barra de herramientas principal de Excel. Esta organizada en pestañas, cada una con grupos de comandos relacionados.

Pestañas principales de la cinta:

  1. INICIO:
    • Portapapeles: Cortar, copiar, pegar, formato de pintor
    • Fuente: Tipo de letra, tamano, negrita, cursiva, color, bordes
    • Alineacion: Alinear texto, combinar celdas, orientacion
    • Numero: Formato de numeros (moneda, porcentaje, fecha, etc.)
    • Estilos: Formato condicional, dar formato como tabla, estilos de celda
    • Celdas: Insertar, eliminar, formato de celdas
    • Edicion: Autosuma, rellenar, ordenar y filtrar, buscar y seleccionar
  2. INSERTAR:
    • Tablas, ilustraciones, graficos, minigraficos, filtros, enlaces
    • Texto (cuadros de texto, encabezado y pie de pagina, objetos)
    • Simbolos, ecuaciones, graficos SmartArt
  3. DISENO DE PAGINA:
    • Temas, configuracion de pagina, opciones de hoja, organizar
  4. FORMULAS:
    • Biblioteca de funciones, nombres definidos, auditoria de formulas
  5. DATOS:
    • Obtener datos, consultas y conexiones, ordenar y filtrar, herramientas de datos
  6. REVISION:
    • Comentarios, proteccion, control de cambios, comparar libros
  7. VISTA:
    • Vistas de libro, Mostrar, Zoom, Ventana, Macros

Barra de formulas

La barra de formulas se encuentra debajo de la cinta de opciones. Muestra:

  • Cuadro de nombres: Indica la celda activa (ej: A1, B5, C10)
  • Contenido de la celda: Muestra el texto, numero o formula de la celda seleccionada
  • Botones de edicion: Cancelar (X), Introducir (✓), Insertar funcion (fx)

Barra de estado

La barra de estado esta en la parte inferior de la ventana y muestra:

  • Informacion sobre la hoja actual (Listo, Calculando, etc.)
  • Estadisticas de las celdas seleccionadas (promedio, conteo, suma numerica)
  • Controles de zoom (deslizador y porcentaje)
  • Vistas de pagina (Normal, Diseño de pagina, Vista previa de salto de pagina)

📙 Miercoles 17 de Junio - Libros, Hojas de Calculo, Celdas, Filas y Columnas

Estructura basica de los archivos de Excel
Objetivo del tema: Comprender la estructura jerarquica de Excel: libros, hojas, celdas, filas y columnas. Aprender a navegar y seleccionar elementos.

📖 Estructura de un archivo de Excel

Excel organiza la informacion en una estructura jerarquica de elementos. Es importante entender cada nivel para trabajar correctamente.

Jerarquia de elementos en Excel:

  • Libro (Workbook): Es el archivo completo de Excel (.xlsx). Puede contener multiples hojas.
  • Hoja de calculo (Worksheet): Cada pagina o pestaña dentro del libro. Contiene filas y columnas.
  • Fila (Row): Linea horizontal de celdas. Se identifican con numeros (1, 2, 3, 4...).
  • Columna (Column): Linea vertical de celdas. Se identifican con letras (A, B, C, D...).
  • Celda (Cell): La interseccion de una fila y una columna. Es la unidad basica donde se ingresan datos.
  • Rango (Range): Conjunto de celdas seleccionadas (ej: A1:C5).

Referencia de celda: Cada celda tiene una direccion unica compuesta por la letra de la columna y el numero de la fila. Por ejemplo:

  • A1 = Columna A, Fila 1
  • B5 = Columna B, Fila 5
  • C10 = Columna C, Fila 10
  • AA1 = Columna AA (columna 27), Fila 1

Limites de Excel

Excel tiene limites importantes que debes conocer:

Elemento Limite en Excel
Filas por hoja 1,048,576 (mas de 1 millon)
Columnas por hoja 16,384 (de A hasta XFD)
Celdas por hoja 17,179,869,184 (17 mil millones aprox.)
Hojas por libro Limite determinado por la memoria de la computadora
Caracteres por celda 32,767 caracteres

Navegacion en Excel

Para moverte por la hoja de calculo puedes usar:

  • Teclas de direccion: Flechas arriba, abajo, izquierda, derecha
  • Tecla Enter: Baja una celda
  • Tecla Tab: Avanza una celda a la derecha
  • Ctrl + Inicio: Va a la celda A1
  • Ctrl + Fin: Va a la ultima celda con datos
  • Ctrl + Flecha: Salta al borde del bloque de datos
  • Barra de desplazamiento: Navega con el mouse
  • Cuadro de nombres: Escribe la referencia y presiona Enter para ir directamente

Seleccion de celdas, filas y columnas

  • Una celda: Haz clic en ella
  • Rango de celdas: Haz clic y arrastra, o haz clic en la primera y Shift + clic en la ultima
  • Fila completa: Haz clic en el numero de la fila
  • Columna completa: Haz clic en la letra de la columna
  • Toda la hoja: Haz clic en el boton de interseccion (esquina superior izquierda)
  • Celdas no contiguas: Selecciona la primera, luego Ctrl + clic en las demas

Tipos de datos que se pueden ingresar en una celda

  1. Texto: Letras, palabras, frases. Se alinean a la izquierda automaticamente.
  2. Numeros: Valores numericos. Se alinean a la derecha automaticamente.
  3. Fechas y horas: Excel las trata como numeros especiales.
  4. Formulas: Comienzan con el signo igual (=). Realizan calculos.
  5. Funciones: Formulas predefinidas (ej: =SUMA(A1:A5)).

📓 COPIAR EN CUADERNO - Tema 4

Como guardar un archivo en Excel

  1. Guardar por primera vez:
    • Ve a ARCHIVO → Guardar como (o presiona F12)
    • Selecciona la ubicacion (Este equipo, OneDrive, etc.)
    • Escribe el nombre del archivo
    • En "Tipo" selecciona el formato deseado (.xlsx, .xls, .csv, .pdf)
    • Haz clic en Guardar
  2. Guardar cambios:
    • Presiona Ctrl + G (o Ctrl + S en algunas versiones)
    • O haz clic en el icono de disquete en la barra de acceso rapido
    • O ve a ARCHIVO → Guardar
  3. Guardar como copia:
    • Ve a ARCHIVO → Guardar como
    • Cambia el nombre o la ubicacion
    • Selecciona un formato diferente si es necesario
    • Haz clic en Guardar

Como abrir un archivo existente

  1. Ve a ARCHIVO → Abrir (o presiona Ctrl + A)
  2. Selecciona la ubicacion donde esta guardado el archivo
  3. Navega por las carpetas y selecciona el archivo
  4. Haz clic en Abrir

Atajo rapido: Presiona Ctrl + A para abrir el cuadro de dialogo Abrir archivo.

Diferencias entre .xlsx y .xls

  • .xlsx es el formato moderno (Excel 2007 en adelante). Los archivos son mas pequenos y seguros.
  • .xls es el formato antiguo (Excel 97-2003). Los archivos son mas grandes y pueden tener limitaciones.
  • Al guardar en .xls, algunas caracteristicas modernas se pierden (graficos avanzados, formato condicional complejo, etc.)

Exportar a PDF

  1. Ve a ARCHIVO → Guardar como
  2. Selecciona la ubicacion
  3. En "Tipo", selecciona PDF
  4. Haz clic en Opciones para configurar:
    • Guardar todo el libro o solo hojas activas
    • Rango de paginas a exportar
  5. Haz clic en Guardar

Alternativa rapida: Ve a ARCHIVO → Exportar → Crear documento PDF/XPS.

El formato CSV pierde formato: Al guardar como CSV, se pierden colores, bordes, formulas y formato. Solo se conserva el texto y los numeros. Usa CSV solo para intercambiar datos crudos.

📖 Operaciones basicas con hojas de calculo

El manejo de hojas de calculo incluye todas las operaciones que podemos realizar para organizar y modificar la estructura de nuestro libro de Excel.

Operaciones con hojas:

  • Insertar hoja: Agregar una nueva hoja al libro
  • Eliminar hoja: Borrar una hoja existente
  • Renombrar hoja: Cambiar el nombre de una hoja
  • Mover hoja: Cambiar la posicion de una hoja dentro del libro
  • Copiar hoja: Duplicar una hoja
  • Cambiar color de pestaña: Asignar un color a la pestaña de la hoja
  • Ocultar/Mostrar hoja: Ocultar hojas que no se quieren mostrar

Operaciones con filas y columnas:

  • Insertar fila/columna: Agregar una nueva fila o columna
  • Eliminar fila/columna: Borrar una fila o columna existente
  • Ajustar ancho/alto: Modificar el tamano de columnas y filas
  • Ocultar/Mostrar: Ocultar o mostrar filas y columnas

📓 COPIAR EN CUADERNO - Tema 5

Gestion de hojas de calculo

  1. Insertar una nueva hoja:
    • Haz clic en el boton + junto a las pestañas de hojas
    • O haz clic derecho en una pestaña → Insertar → Hoja
  2. Eliminar una hoja:
    • Haz clic derecho en la pestaña → Eliminar
    • Confirma la eliminacion (¡los datos se pierden permanentemente!)
  3. Renombrar una hoja:
    • Haz doble clic en la pestaña de la hoja
    • O haz clic derecho → Cambiar nombre
    • Escribe el nuevo nombre y presiona Enter
  4. Mover o copiar una hoja:
    • Haz clic derecho en la pestaña → Mover o copiar
    • Selecciona la posicion donde quieres moverla
    • Marca Crear una copia si quieres duplicarla
    • Haz clic en Aceptar
  5. Cambiar color de pestaña:
    • Haz clic derecho en la pestaña → Color de etiqueta
    • Selecciona un color de la paleta

Gestion de filas y columnas

  1. Insertar una fila:
    • Selecciona la fila donde quieres insertar (haz clic en el numero)
    • Haz clic derecho → Insertar
    • Se inserta una fila nueva arriba de la seleccionada
  2. Insertar una columna:
    • Selecciona la columna donde quieres insertar (haz clic en la letra)
    • Haz clic derecho → Insertar
    • Se inserta una columna nueva a la izquierda de la seleccionada
  3. Eliminar fila o columna:
    • Selecciona la fila o columna
    • Haz clic derecho → Eliminar
  4. Ajustar ancho de columna:
    • Arrastra el borde derecho del encabezado de la columna
    • O haz doble clic en el borde para ajuste automatico
    • O haz clic derecho → Ancho de columna → escribe el valor
  5. Ajustar alto de fila:
    • Arrastra el borde inferior del encabezado de la fila
    • O haz doble clic para ajuste automatico
    • O haz clic derecho → Alto de fila → escribe el valor

Operaciones con celdas

  • Copiar celda: Selecciona → Ctrl + C → selecciona destino → Ctrl + V
  • Cortar celda: Selecciona → Ctrl + X → selecciona destino → Ctrl + V
  • Eliminar contenido: Selecciona → Supr o Delete
  • Deshacer: Ctrl + Z
  • Rehacer: Ctrl + Y

📕 Jueves 18 de Junio - Guardado y Apertura de Archivos (.xlsx, .xls, .csv, .pdf)

Formatos de archivo y metodos de guardado en Excel
Objetivo del tema: Aprender a guardar y abrir archivos de Excel en diferentes formatos. Conocer las ventajas y desventajas de cada formato.

📖 Formatos de archivo en Excel

Excel permite guardar archivos en multiples formatos. Cada formato tiene caracteristicas especificas que lo hacen adecuado para diferentes situaciones.

Formato Extension Descripcion Uso recomendado
Libro de Excel .xlsx Formato predeterminado de Excel 2007 y posteriores. Usa compresion XML. Trabajo diario, archivos con formulas y formato
Libro de Excel 97-2003 .xls Formato antiguo de Excel. Compatible con versiones anteriores. Compartir con usuarios de Excel muy antiguo
CSV .csv Valores separados por comas. Solo texto y numeros, sin formato. Importar/exportar datos entre programas
PDF .pdf Formato de documento portable. No editable, solo para lectura. Compartir informes finales, imprimir, distribuir
Libro habilitado para macros .xlsm Igual que .xlsx pero permite guardar macros (programacion). Archivos con automatizaciones y macros
Libro binario .xlsb Formato binario mas compacto y rapido que .xlsx Archivos muy grandes que necesitan abrirse rapido

📓 COPIAR EN CUADERNO - Tema 4

Como guardar un archivo en Excel

  1. Guardar por primera vez:
    • Ve a ARCHIVO → Guardar como (o presiona F12)
    • Selecciona la ubicacion (Este equipo, OneDrive, etc.)
    • Escribe el nombre del archivo
    • En "Tipo" selecciona el formato deseado (.xlsx, .xls, .csv, .pdf)
    • Haz clic en Guardar
  2. Guardar cambios:
    • Presiona Ctrl + G (o Ctrl + S en algunas versiones)
    • O haz clic en el icono de disquete en la barra de acceso rapido
    • O ve a ARCHIVO → Guardar
  3. Guardar como copia:
    • Ve a ARCHIVO → Guardar como
    • Cambia el nombre o la ubicacion
    • Selecciona un formato diferente si es necesario
    • Haz clic en Guardar

Como abrir un archivo existente

  1. Ve a ARCHIVO → Abrir (o presiona Ctrl + A)
  2. Selecciona la ubicacion donde esta guardado el archivo
  3. Navega por las carpetas y selecciona el archivo
  4. Haz clic en Abrir

Diferencias entre .xlsx y .xls

  • .xlsx es el formato moderno (Excel 2007 en adelante). Los archivos son mas pequenos y seguros.
  • .xls es el formato antiguo (Excel 97-2003). Los archivos son mas grandes y pueden tener limitaciones.
  • Al guardar en .xls, algunas caracteristicas modernas se pierden (graficos avanzados, formato condicional complejo, etc.)

Exportar a PDF

  1. Ve a ARCHIVO → Guardar como
  2. Selecciona la ubicacion
  3. En "Tipo", selecciona PDF
  4. Haz clic en Opciones para configurar:
    • Guardar todo el libro o solo hojas activas
    • Rango de paginas a exportar
  5. Haz clic en Guardar

Alternativa rapida: Ve a ARCHIVO → Exportar → Crear documento PDF/XPS.

💡 CONSEJO IMPORTANTE - Tema 4

Guarda frecuentemente: Excel tiene autoguardado en versiones modernas, pero es buena practica guardar manualmente cada pocos minutos. Usa Ctrl + G como reflejo.

Nombres descriptivos: Usa nombres de archivo que indiquen el contenido y la fecha. Ejemplo: "Notas_Matematicas_2026.xlsx" en lugar de "Libro1.xlsx".

El formato CSV pierde formato: Al guardar como CSV, se pierden colores, bordes, formulas y formato. Solo se conserva el texto y los numeros. Usa CSV solo para intercambiar datos crudos.

📓 Viernes 19 de Junio - Manejo Basico de Hojas de Calculo

Operaciones fundamentales con hojas, filas, columnas y celdas
Objetivo del tema: Aprender a insertar, eliminar, renombrar, mover y copiar hojas. Manejar filas, columnas y celdas de manera eficiente.

📖 Operaciones basicas con hojas de calculo

El manejo de hojas de calculo incluye todas las operaciones que podemos realizar para organizar y modificar la estructura de nuestro libro de Excel.

Operaciones con hojas:

  • Insertar hoja: Agregar una nueva hoja al libro
  • Eliminar hoja: Borrar una hoja existente
  • Renombrar hoja: Cambiar el nombre de una hoja
  • Mover hoja: Cambiar la posicion de una hoja dentro del libro
  • Copiar hoja: Duplicar una hoja
  • Cambiar color de pestaña: Asignar un color a la pestaña de la hoja
  • Ocultar/Mostrar hoja: Ocultar hojas que no se quieren mostrar

Operaciones con filas y columnas:

  • Insertar fila/columna: Agregar una nueva fila o columna
  • Eliminar fila/columna: Borrar una fila o columna existente
  • Ajustar ancho/alto: Modificar el tamano de columnas y filas
  • Ocultar/Mostrar: Ocultar o mostrar filas y columnas

📓 COPIAR EN CUADERNO - Tema 5

Gestion de hojas de calculo

  1. Insertar una nueva hoja:
    • Haz clic en el boton + junto a las pestañas de hojas
    • O haz clic derecho en una pestaña → Insertar → Hoja
  2. Eliminar una hoja:
    • Haz clic derecho en la pestaña → Eliminar
    • Confirma la eliminacion (los datos se pierden permanentemente)
  3. Renombrar una hoja:
    • Haz doble clic en la pestaña de la hoja
    • O haz clic derecho → Cambiar nombre
    • Escribe el nuevo nombre y presiona Enter
  4. Mover o copiar una hoja:
    • Haz clic derecho en la pestaña → Mover o copiar
    • Selecciona la posicion donde quieres moverla
    • Marca Crear una copia si quieres duplicarla
    • Haz clic en Aceptar
  5. Cambiar color de pestaña:
    • Haz clic derecho en la pestaña → Color de etiqueta
    • Selecciona un color de la paleta

Gestion de filas y columnas

  1. Insertar una fila:
    • Selecciona la fila donde quieres insertar (haz clic en el numero)
    • Haz clic derecho → Insertar
    • Se inserta una fila nueva arriba de la seleccionada
  2. Insertar una columna:
    • Selecciona la columna donde quieres insertar (haz clic en la letra)
    • Haz clic derecho → Insertar
    • Se inserta una columna nueva a la izquierda de la seleccionada
  3. Eliminar fila o columna:
    • Selecciona la fila o columna
    • Haz clic derecho → Eliminar
  4. Ajustar ancho de columna:
    • Arrastra el borde derecho del encabezado de la columna
    • O haz doble clic en el borde para ajuste automatico
    • O haz clic derecho → Ancho de columna → escribe el valor
  5. Ajustar alto de fila:
    • Arrastra el borde inferior del encabezado de la fila
    • O haz doble clic para ajuste automatico
    • O haz clic derecho → Alto de fila → escribe el valor

Operaciones con celdas

  • Copiar celda: Selecciona → Ctrl + C → selecciona destino → Ctrl + V
  • Cortar celda: Selecciona → Ctrl + X → selecciona destino → Ctrl + V
  • Eliminar contenido: Selecciona → Supr o Delete
  • Deshacer: Ctrl + Z
  • Rehacer: Ctrl + Y

Ejercicios practicos:

  5. Inserta una fila entre la fila 2 y 3
  6. Inserta una columna entre la columna B y C
  7. Ajusta el ancho de todas las columnas automaticamente
  8. Copia la hoja "Datos" y renombra la copia como "Backup"
  9. Mueve la hoja "Backup" al final de todas las hojas
  10. Oculta la hoja "Auxiliar" y luego muestrala de nuevo

💡 CONSEJO IMPORTANTE - Tema 5

Organiza tus hojas: Usa nombres descriptivos para las hojas. Evita nombres como "Hoja1", "Hoja2". Mejores nombres: "Ventas_Enero", "Clientes", "Inventario".

Colores de pestaña: Usa colores para categorizar. Por ejemplo: verde para hojas de datos, azul para calculos, rojo para alertas.

Cuidado al eliminar: Al eliminar una hoja, los datos se pierden permanentemente. Si no estas seguro, oculta la hoja en lugar de eliminarla.

Ajuste automatico: Haz doble clic en el borde entre dos encabezados de columna para ajustar el ancho automaticamente al contenido. Es la forma mas rapida de hacer que todo se vea bien.
